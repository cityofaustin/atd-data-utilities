# atd-jobs-util (jobutil)
Python utilities for registering postgREST scripting [jobs](https://github.com/cityofaustin/atd-data-deploy/).

## Installation

Master branch:

```
pip install atd-jobs-util
```

Development branch:

```
pip install atd-jobs-util-dev
```
